This is the 2PrintBeta DICE_TC Arduino Library - Using software SPI for Arduino Due

These modules use SPI to communicate, 3 pins are required to  
interface

Written by Dominik Wenger for 2PrintBeta.

Based on Adafruit_MAX31885 code by Limor Fried/Ladyada  for Adafruit Industries.  
BSD license, check license.txt for more information

To download. click the DOWNLOADS button in the top right corner, rename the uncompressed folder DICE_TC_lib. Check that the DICE_TC_lib folder contains DICE_TC_lib.cpp and DICE_TC_lib.h

Place the DICE_TC_lib library folder your <arduinosketchfolder>/libraries/ folder. You may need to create the libraries subfolder if its your first library. Restart the IDE.
